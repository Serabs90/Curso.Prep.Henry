Variables: se utiliza para darle un "valor" a "algo", es una forma de definir un tipo de dato (string, number o booleano) y asignarle un valor. Si se define como var el valor puede modificarse luego, si se define como const su valor se declara al principio y no se puede modificar luego.
Strings: es una "cadena" de caracteres, se define entre comillas ' ' o " ". Generalmente son letras, aun que puede ser cualquier tipo de dato que dentro de las comillas se convierte en string.
Funciones: Las funciones son un tipo de objeto, "llamable" o invocable, que nos permiten trabajar con las variables. Se componen con la palabra clave function, el nombre que se le asigna a la función, puede contener argumentos (que son variables dentro de la función y se definen dentro de parentesis ()) y los corchetes {} que es donde va el código de la función. La función devuelve por defecto "undefined", si nosotros queremos que nos devuelva el resultado de la operación que realiza el código devemos declarar un return, que es la forma de decirle a la función que nos muestre el resultado fuera de ella. Sólo puede haber UN return dentro de la función, ya que aquí deja de ejecutar el código de la función, si hay algo después del return no lo va a leer ni ejecutar.
Declaraciones `if`: if equivale a 'si', es un operador de control de flujo dentro de la función que valida la variable, si es verdadera ejecuta el código que le asignamos o si es falsa sigue avanzando.
Valores booleanos (`true`, `false`): los valores booleanos se usan para declarar como verdadero o falso una variable o un tipo de dato dentro de ella. Usualmente también realizan una comparación de 2 variables u objetos y devuelven verdadero o falso como definición
